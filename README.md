# PennApps17-backend
Back end for PennApps F17 project

## Setup instructions

To setup `virtualenv` with Python 3.x, navigate to the root directory of this repo and type:

```
$ virtualenv -p python3 .
```

Activate `virtualenv`:
```
$ source bin/activate
```

Deactivate `virtualenv`:
```
$ deactivate
```
